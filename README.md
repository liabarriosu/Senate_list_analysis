# Paraguayan Senate public officials Data Cleaning & Analysis

This repository contains a script to clean, structure, and analyze the list of staff members from the Paraguayan Senate.
The workflow is designed to be reusable: journalists only need to upload the monthly dataset they want to analyze, and the script will process it without any modifications.

## This what the code does

- Cleans and standardizes staff data
- Removes duplicates and inconsistencies
- Structures the dataset for analysis
- Outputs a clean table
- Create a lots of tables with numbers ready to use for reporting

## How to use it?

1. Download or export the monthly staff list from the Senate
2. Upload it to the project environment in google colab
3. Run the script

The output is a clean dataset that can be used for month-to-month comparison and journalistic analysis.
